# 《不许跳！》关卡灰盒纲要

## 通用规则

- 每个房间只回答一个问题：“玩家在这里要学会或证明什么？”
- 新机制第一次安全，第二次要求应用，第三次才允许明显损失。
- 普通路线宽容，捷径承担精度与速度压力。
- 房间失败后 1.5 秒内恢复操作。

[→ level-design：介绍—发展—变化；Miyamoto：关卡即教学]

## 0. 入职校准

| 房间 | 命题 | 保护 |
|---|---|---|
| 0-1 | 枪口朝下，角色向上 | 软垫地面，无死亡 |
| 0-2 | 斜向开火改变飞行方向 | 左右错层宽平台 |
| 0-3 | 空中一发可以救险 | 下方大型救援台 |

结尾广播：“恭喜，你已证明枪比跳跃安全。”作为首个笑点和节奏谷值。

## 1. 精准弹射科

| 房间 | 普通路线 | 捷径 |
|---|---|---|
| 1-1 预判抛物线 | 三个宽度递减的平台 | 一次斜射跨两级 |
| 1-2 控制水平速度 | 墙面缓冲＋中间平台 | 擦墙反弹直达出口 |
| 1-3 连续落点 | 三个节拍式平台 | 两枪连续飞越 |

[→ level-design：由单点技巧到节奏组合；Raph Koster：掌握带来乐趣]

## 2. 大力出奇迹科

| 房间 | 普通路线 | 捷径 |
|---|---|---|
| 2-1 认识霰弹大冲量 | 封闭安全试射室 | 无，先让玩家玩 |
| 2-2 跨越大缺口 | 两段跳板路线 | 一枪跨越全缺口 |
| 2-3 提前制动 | 远端缓冲墙＋落地区 | 借墙反弹进出口 |

霰弹画面可以散射，但角色冲量必须固定。[→ game-design-fundamentals：结果可学习；Jonathan Blow：机制诚实]

## 3. 空中姿态科

| 房间 | 命题 | 保护 |
|---|---|---|
| 3-1 三次小冲量形成曲线 | 绕过静态弧形障碍 | 下方救援平台 |
| 3-2 观察移动平台周期 | 单个慢平台 | 安全等待区 |
| 3-3 曲线与平台组合 | 两个平台错相移动 | 中段重试 |

区域结尾安排平坦走廊和短广播，降低强度后再进入毕业测试。[→ level-design：高潮前需要谷值；Jenova Chen：节奏释放]

## 4. 毕业测试

1. 手枪段：两个窄平台，快速重温精准角度。
2. 霰弹段：大缺口；下方有耗时但安全的回收传送带。
3. 三连发段：绕旋转挡板；命中只弹回，不直接死亡。
4. 火箭筒段：三段稳妥路线和一炮捷径同时可见。
5. 出口段：倒计时停止、门禁卡笑点、成绩结算。

若工期不足，火箭筒改为固定力度的一次性发射台，不建立完整武器系统。[→ game-design-fundamentals：控制范围；Fumito Ueda：先删昂贵边缘功能]
